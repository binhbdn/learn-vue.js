Navigation Guards

Per-Route Guard

You can define beforeEnter guards directly on a route's configuration object:

        const router = new VueRouter({
            routes: [
                {
                    path: '/foo',
                    component: Foo,
                    beforeEnter: (to, from, next) => {
                    // ...
                    }
                }
            ]
        })

These guards have the exact same signature as global before guards.
