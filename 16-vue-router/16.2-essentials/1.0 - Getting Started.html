Getting Started

Create SPA with Vue + Vue Router

  • Vue.js : compose our app with components
  • Vue Router : map our components to the routes and let Vue Router know where to render them

inside of any component:

  • router : this.$router
  • current route : this.$route
        JS in : Home.vue
        export default {
          computed: {
            username() {
              // We will see what `params` is shortly
              return this.$route.params.username
            }
          },
          methods: {
            goBack() {
              window.history.length > 1 ? this.$router.go(-1) : this.$router.push('/')
            }
          }
        }

Notice that a <router-link> automatically gets the .router-link-active class when its target route is matched. You can learn more about it in its API reference.
