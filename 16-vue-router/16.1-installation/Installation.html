Installation

Direct Download / CDN

  • latest release on npm:
    https://unpkg.com/vue-router/dist/vue-router.js
  • use a specific version/tag via URLs like:
    https://unpkg.com/vue-router@2.0.0/dist/vue-router.js

Include vue-router after Vue and it will install itself automatically:

        <script src="/path/to/vue.js"></script>
        <script src="/path/to/vue-router.js"></script>

npm

        npm install vue-router

When used with a module system, you must explicitly install the router via Vue.use():

        import Vue from 'vue'
        import VueRouter from 'vue-router'
        
        Vue.use(VueRouter)

You don't need to do this when using global script tags.

Vue CLI

        vue add router
  • add Vue Router as a plugin
  • generate global script tags
  • generate two sample routes
  • overwrite your App.vue

Dev Build

Clone from GitHub and build vue-router yourself

        git clone https://github.com/vuejs/vue-router.git node_modules/vue-router
        cd node_modules/vue-router
        npm install
        npm run build
