Using vuex in builds

When building a Webcomponent or Library, the entry point is not main.js, but an entry-wc.js file, generated here: https://github.com/vuejs/vue-cli/blob/dev/packages/%40vue/cli-service/lib/commands/build/resolveWcEntry.js

So to use vuex in web component target, you need to initialize the store in App.vue:

        import store from './store'

        // ...
        
        export default {
          store,
          name: 'App',
          // ...
        }
