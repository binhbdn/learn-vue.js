Installation

  • install
    • npm install -g @vue/cli
    • yarn global add @vue/cli
  • check
    • vue
    • vue --version
  • upgrade global Vue CLI installation
    • npm update -g @vue/cli
    • yarn global upgrade --latest @vue/cli
  • upgrade related packages (including packages starting with @vue/cli-plugin- or vue-cli-plugin-) inside your project
    • run vue upgrade inside the project directory
