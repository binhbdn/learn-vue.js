Overview

  • Cũ : vue-cli
  • Mới : @vue/cli
  • Dùng @vue/cli để tạo project scaffolding.
  • Dùng @vue/cli + @vue/cli-service-global để tạo nhanh prototyping không cần cấu hình.
  • Runtime dependency (@vue/cli-service):
    • Có thể nâng cấp;
    • Built on top of webpack, với các giá trị mặc định (defaults) hợp lý;
    • Có thể định cấu hình thông qua tệp cấu hình trong dự án;
    • Có thể mở rộng thông qua các plugin
  • Một bộ sưu tập phong phú các plugin chính thức được tích hợp các công cụ tốt nhất trong hệ sinh thái giao diện người dùng.
  • Giao diện người dùng đồ họa đầy đủ để tạo và quản lý các dự án Vue.js.

Components of the System

CLI @vue/cli

  • a globally installed npm package
  • vue create : quickly scaffold a new project
  • vue serve : instantly prototype new ideas
  • vue ui : manage your projects using a graphical user interface
  • ...

CLI Service @vue/cli-service

  • a development dependency
  • an npm package installed locally into every project created by @vue/cli
  • is built on top of webpack and webpack-dev-server
    • core service : loads other CLI Plugins
    • An internal webpack config that is optimized for most apps
    • The vue-cli-service binary inside the project, which comes with the basic serve, build and inspect commands.

CLI Plugins

  • are npm packages that provide optional features to your Vue CLI projects, such as Babel/TypeScript transpilation, ESLint integration, unit testing, and end-to-end testing...
  • names start with either @vue/cli-plugin- (for built-in plugins) or vue-cli-plugin- (for community plugins)
  • run vue-cli-service : it automatically resolves and loads all CLI Plugins listed in your project's package.json
  • can be included as part of your project creation process or added into the project later
  • can also be grouped into reusable presets
