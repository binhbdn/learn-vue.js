Chúng ta có thể truyền một mảng vào v-bind:class để áp dụng một danh sách class.

Nếu muốn bật tắt theo điều kiện một class trong danh sách, bạn có thể dùng một toán tử ba ngôi (ternary expression). Đoạn code này sẽ luôn luôn áp dụng class errorClass, nhưng chỉ áp dụng bgGreyClass khi isBgGrey mang giá trị đúng.

Cách làm trên có thể hơi dài dòng nếu bạn có nhiều class theo điều kiện. Bạn có thể dùng cú pháp ngắn gọn hơn là dùng object bên trong cú pháp mảng.
