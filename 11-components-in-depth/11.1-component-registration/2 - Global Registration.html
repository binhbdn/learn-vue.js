Global Component Registration / Đăng kí component ở cấp toàn cục

Để đăng kí component ở cấp toàn cục, hãy sử dụng cú pháp Vue.component:

        Vue.component('my-component-name', {
            // ... các tuỳ biến ...
        })

Các component được đăng kí ở cấp toàn cục có thể sử dụng trong template của bất cứ đối tượng Vue gốc (new Vue) nào. Ví dụ:

        JS:
        Vue.component('component-a', { /* ... */ })
        Vue.component('component-b', { /* ... */ })
        Vue.component('component-c', { /* ... */ })
        
        new Vue({ el: '#app' })


        HTML:
        <div id="app">
            <component-a></component-a>
            <component-b></component-b>
            <component-c></component-c>
        </div>

Điều này còn áp dụng cho tất cả những components con, tức là tất cả ba component trên đều có thể sử dụng lẫn nhau.
