In addition to data properties, Vue instances expose a number of useful instance properties and methods. These are prefixed with $ to differentiate them from user-defined properties.

Ngoài data, một đối tượng Vue cũng hỗ trợ một số thuộc tính và phương thức đối tượng (instance properties & methods) hữu dụng khác. Các thuộc tính và phương thức này được bắt đầu bằng kí hiệu $ để phân biệt với thuộc tính và phương thức do người dùng định nghĩa.

{{a}} render to {{a}}

    // Test on console

    vm.$data === data // => true
    vm.$el === document.getElementById('example') // => true
    
    // $watch is an instance method
    // $watch là một phương thức của đối tượng Vue
    vm.$watch('a', function (newValue, oldValue) {
        // This callback will be called when `vm.a` changes
        // Hàm callback này sẽ được gọi khi `vm.a` thay đổi
        alert('a changed from ' + oldValue + ' to ' + newValue);
    })
