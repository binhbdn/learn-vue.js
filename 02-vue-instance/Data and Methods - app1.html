When a Vue instance is created, it adds all the properties found in its data object to Vue’s reactivity system. When the values of those properties change, the view will “react”, updating to match the new values.

When this data changes, the view will re-render. Khi dữ liệu thay đổi, view sẽ render lại.

{{a}} render to {{a}}

{{b}} render to {{b}}

    // Test on console

    // Getting the property on the instance
    // returns the one from the original data
    // Truy xuất đến thuộc tính của đối tượng 
    // trả về giá trị của object "data" đã khởi tạo 
    vm.a == data.a // => true

    // Setting the property on the instance
    // also affects the original data
    // Thay đổi thuộc tính của vm cũng
    // ảnh hưởng đến dữ liệu ban đầu
    vm.a = 2
    data.a // => 2

    // ... and vice-versa
    // ... và ngược lại
    data.a = 3
    vm.a // => 3

    // not trigger any view updates
    // không kích hoạt thay đổi trên view
    data.b = 4
